This website displays updated caseload information on the corona virus in the USA and analyzes the impact of the corona virus on various aspects of life and society. Four key impacts are analyzed: crime, unemployment, internet usage and stocks. State level data is pulled from the COVID Tracking Project's API updated daily (main data source: state public health authorities). County level data is pulled from the Coronavirus Tracker API which updated throughout the day (main data source: Conference of State Bank Supervisors (CSBS)).
